# react-spoiler
A spoiler component for reactjs

### Install
using yarn  
`yard add react-spoiler`

using npm  
`npm install react-spoiler`


## Usage

`import ReactSpoiler from "react-spoiler";`


```
<ReactSpoiler>
  <h1>ReactSpoiler</h1>
  <p> Click/Hover me to the magic! </p>
</ReactSpoiler>
```
